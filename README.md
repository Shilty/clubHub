# clubHub

**clubHub** is a web application for managing student clubs and campus activities. It provides a complete ecosystem for students, club leaders, and platform administrators to interact, organize events, and share announcements.

## Features

- **Guest / Student**
  - View clubs and announcements.
  - Submit requests to join clubs.
  - View and participate in club events.
- **Club Admin / Leader**
  - Create, edit, and delete clubs, events, and announcements.
  - Manage members and roles within their club.
- **Platform Admin**
  - Approve new clubs.
  - Moderate communities and chat rooms.
  - Manage users and accounts.
- **Extra**
  - Chat room for each club to enable communication among members.

## Tech Stack

- **Frontend:** React, TypeScript, TailwindCSS
- **Backend:** .NET 7 Web API
- **Database:** Microsoft SQL server
